<h1>🔒Validador de Senha Segura</h1>

Este é um projeto em Python focado em **CyberSegurança** e práticas de **Desenvolvimento Seguro (Secure Coding)**. 

O programa utiliza Expressões Regulares (Regex) avançadas para criar uma validação multicritério, exigindo obrigatoriamente que a senha criada possua:
* No mínimo 8 caracteres;
* Pelo menos 1 caractere numérico;
* Pelo menos 1 caractere especial (como @, $, !, %, *, ?, &,#).
* Pelo menos 1 letra maiúscula;
* Pelo menos 1 letra minúscula;

Além disso, o sistema conta com uma lógica de **Mascaramento de Dados**, substituindo os caracteres reais por asteriscos para simular a proteção de privacidade em telas e logs do sistema.
